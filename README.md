## what is box-office?
box-office is a Sinatra app that uses the [OMDB API](http://www.omdbapi.com).

----
##installation
- `git clone git@github.com:LARailsLearners/box-office.git`
- `ruby box-office.rb`

---
##development
- assign yourself to issue on
